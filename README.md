### User section
#### Describes the program for users, that is - players of the game. 
No name is a text-based, RPG game coded in C++. The concept...

User starts as a player with 3 moves:
- Punch (20 HP damage)
- Block (blocks 10 HP incoming damage)
- Heal (heals 40 HP)

You will face 4 different enemies - Minion, Two headed giant, Dauthi and a Hydra (has 2 stages) all with different moves.
|Minion      |Giant         |Dauthi        |Hydra (stage 1)      |Hydra (2 stage)      |
|------------|--------------|--------------|---------------------|---------------------|
|Kick        |Sword slice   |Slice         |Fire cough           |Fire Blast           |
|Observe     |Axe Cleave    |Soul sucker   |Stomp                |Electric Breath      |
|Block       |Enrage        |Invisible     |Fire ring            |Decay                |
|Fuck you    |              |              |                     |Element rings        |

You see our health and stamina bar under the window, and when your health reaches 0 you die and lose the game.


To start, run 
```
make
```
then ?
```
./rpg
```
The you see in the terminal the game screen:)
